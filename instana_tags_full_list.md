# Full List of Application Tags from Instana

## Commonly Used
- **Application Name** (`application.name`)
- **Call Erroneous** (`call.erroneous`) - Boolean flag indicating whether the call resulted in an error.
- **Call Type** (`call.type`) - Primary classification of a call based on its communication protocol or interaction pattern.

## Agent
- **Tag** (`agent.tag`) - Custom tag applied to the agent.
- **Zone** (`agent.zone`) - Zone where the agent is running.

## Application
- **Inbound of Application** (`call.inbound_of_application`) - Indicates calls coming into applications.
- **Name** (`application.name`).

## Additional Tags
- **otel.gen_ai.prompt.19.tool_call_id** (`gen_ai.prompt.19.tool_call_id`)
- **otel.verbose** (`verbose`)
- **otel.user_api_key** (`user_api_key`)
- **cloudfoundry.organization.id** (`Organization ID`)
- **otel.gen_ai.prompt.58.content** (`gen_ai.prompt.58.content`)
- **kubernetes.namespace.label** (`Label`) - Logical partition within the Kubernetes cluster where the workload is deployed.
- **kubernetes.replicationcontroller.label** (`Label`).
